## Descriere

Acest proiect reprezintă un **website personal de tip CV/portofoliu**, realizat folosind **HTML, CSS și JavaScript**. Scopul proiectului este prezentarea informațiilor personale, a abilităților, a proiectelor realizate și a unor elemente interactive precum o pagină de overview, o carte de vizită digitală și un chatbot simplu.

Website-ul este structurat în mai multe pagini interconectate și oferă o experiență simplă, clară și plăcută de navigare.

---

## Obiective

- Crearea unui CV online
- Prezentarea informațiilor personale
- Prezentarea proiectelor personale
- Realizarea unei cărți de vizită digitale
- Implementarea unei pagini de overview
- Crearea unui chatbot simplu folosind JavaScript
- Exersarea tehnologiilor HTML, CSS și JavaScript
- Realizarea unei interfețe web responsive de bază
- Aplicarea unui design inspirat dintr-un layout realizat în Figma
